# Security Policy

## Supported Versions

The following versions of Bulk-Migrator are currently receiving security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.4.x   | :white_check_mark: |
| < 2.4   | :x:                |

Only the latest minor release (2.4.x) is actively maintained with security patches. Older versions do not receive security updates; users are encouraged to upgrade to the latest release.

## Reporting a Vulnerability

To report a security vulnerability in Bulk-Migrator, please use GitHub's private vulnerability reporting feature:

1. Navigate to the [Security tab](https://github.com/scottlz0310/Bulk-Migrator/security) of this repository.
2. Click **"Report a vulnerability"** to open a private advisory.
3. Provide a clear description of the vulnerability, steps to reproduce, and any potential impact.

**Do not open a public issue for security vulnerabilities.**

- **Initial response**: You can expect an acknowledgement within 7 days of submission.
- **If confirmed**: A patch will be developed and released as soon as possible. You will be credited in the release notes unless you prefer to remain anonymous.
- **If declined**: You will receive an explanation of why the report was not accepted.
