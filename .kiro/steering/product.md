# プロダクト概要

## Bulk-Migration ツール

組織用 OneDrive から SharePoint Online への大容量ファイル移行を自動化するバッチツール

### 主要目的
- OneDrive から SharePoint へ数百GB規模のファイルを安全・効率的に移行
- 転送中にディレクトリ構造を保持
- Microsoft Graph API を利用した 4MB 以上の大容量ファイルのチャンクアップロード対応
- ログ・インデックス・スキップリスト・自動リトライ・進捗監視・watchdog 自動再起動機能による堅牢な運用を提供

### 対象ユーザー
同一組織内で SharePoint サイトと OneDrive を運用している管理者、またはその権限を有するユーザー

### 主要機能
- OneDrive/SharePoint 間でのファイル・フォルダ構造の再帰的転送
- 4MB 以上のファイルに対するチャンクアップロードセッション
- 転送失敗時の自動リトライとスキップリスト管理
- 転送進捗・エラー・成功ログの出力
- watchdog によるフリーズ検知と自動再起動（完了時の未転送ファイルリトライを含む）
- 転送検証用の rebuild/verify バッチ

### 言語・コミュニケーション
- プライマリドキュメントとユーザーインターフェースは日本語
- AI アシスタントとの全ての対話・応答・ドキュメント作成は日本語で実施
- コードコメントも日本語を基本とする（技術的な専門用語は英語併記可）
- 対象ユーザーベースに合わせた日本語中心の開発体制
